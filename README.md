📌 Sistema de Tarefas com Node.js, Express e MySQL

Este projeto consiste no desenvolvimento de um sistema completo de gerenciamento de tarefas (Task Manager), com foco na criação de uma API RESTful integrada a um banco de dados relacional. A aplicação permite realizar operações de CRUD (Create, Read, Update e Delete) de forma eficiente, garantindo organização e persistência dos dados.

O backend foi desenvolvido utilizando Node.js com o framework Express, proporcionando uma estrutura leve e escalável para a construção da API. A comunicação com o banco de dados MySQL foi implementada para armazenar e gerenciar as tarefas de forma segura e consistente.

Além disso, o projeto segue boas práticas de desenvolvimento, incluindo organização de código, separação de responsabilidades e utilização de padrões comuns no desenvolvimento de APIs.

🚀 Funcionalidades...

✅ Criação de novas tarefas
📋 Listagem de tarefas cadastradas
✏️ Atualização de tarefas existentes
❌ Exclusão de tarefas
🔗 Integração com banco de dados MySQL

🛠️ Tecnologias Utilizadas...
Node.js
Express
MySQL
JavaScript (ES6+)
